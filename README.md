# PasswordPal

## Description:

This project is for a password tracking app. The user can enter account/app names (email, streaming service, game account, etc.) and the associated password and PasswordPal can store it in a wallet. Users can come up with a password themselves or have PasswordPal generate a strong password for them! The wallet is able to be viewed at any time and can be edited if the user wishes. Perfect for users who have a hard time remembering passwords, or who want to make sure their passwords are as strong as can be!

## App details:

PasswordPal provides the following functionality:
* Set user-created password
* Set randomly generated password
* View all contents inside wallet
* Find a specific password for a given account. Gives user the option to set account name and password if it is not yet in the wallet.
* Change a password for a saved account in the wallet.

The project uses Java for the backend. Frontend will be implemented soon using Swing GUI.
